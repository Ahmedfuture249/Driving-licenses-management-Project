🚗 Driving License Management System (DVLD)

A desktop application for managing driver licensing operations, including license issuance, renewals, testing, and applicant records.
The system simulates the core services of a Driving & Vehicle License Department.

✨ Features

Driver and applicant management

License issuance, renewal, and replacement

Driving test scheduling and results tracking

International license issuance

License detention and release

Application processing and fee tracking

User and system administration

🏗 Architecture

The system follows a layered architecture:

Presentation Layer — Windows Forms UI

Business Logic Layer — validation and rules

Data Access Layer — SQL Server database operations

🛠 Technologies

C# (.NET WinForms)

SQL Server

ADO.NET

Layered Architecture

🎯 Purpose

To automate licensing processes, enforce eligibility rules, and maintain accurate driver and license records in a structured and efficient system.
